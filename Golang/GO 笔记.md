## 变量声明

### 声明

```go
var x int	//声明一个变量
var x,y int	//声明多个变量
```

### 声明并赋值

```go
var x int = 1	//声明一个变量并赋值
var x,y string = "1","2"	//声明多个变量并赋值

var (
	x int = 3	//声明多个变量并赋值，可以是不同类型
    y int = 4
)
```

### 短声明（类型推断）

```go
x := 1	//声明一个变量
x,y := 1,2	//声明多个变量
//:=左边的变量，必须至少有一个是新定义的
//不能定义全局变量，即函数体外的变量，否
```
